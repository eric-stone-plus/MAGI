# Aufhebung Formal — Hegelian Synthesis Protocol

## Source
QUINTE R3 debate synthesis. Aufhebung must produce genuine dimensional lift, not interpolation average.

## Principle
Aufhebung = thesis-antithesis-synthesis, where synthesis is NOT (thesis+antithesis)/2.
The synthesis must introduce a new variable or system layer that dissolves the contradiction.

## TRIZ Three-Separation Routing

### Time Separation
- Prompt: "Do not average. Construct a temporal sequence: execute perspective A under condition T1, perspective B under condition T2. Define trigger boundaries for T1 and T2."

### Space/Architecture Separation
- Prompt: "Do not compromise. Split the problem into core layer and edge layer. Which layer does perspective A apply to? Which does B apply to? Define the API interface between layers."

### State Transition
- Prompt: "Introduce a third component or change the data's physical form (e.g., sync→async, local→cloud) so that A and B's demands are no longer mutually exclusive."

## Anti-Degradation Mechanisms

### Commitment Tracking
Each thesis/antithesis output must structurally annotate:
- Ontological commitments (what must be true for this to work)
- Inferential commitments (what follows if this is accepted)
- Pragmatic commitments (what actions this implies)

Synthesis must produce a transformation log:
- Preserved commitments
- Negated commitments
- Emergent commitments (new, not in either original)
- Suspended commitments (deferred)

### Destructive Testing
After synthesis: "Is this conclusion merely a compromise between A and B?"
If yes → synthesis rejected, re-run with TRIZ routing.

### Orthogonal Dimension Non-Zero Gain
Synthesis must introduce at least one new variable not present in thesis or antithesis.
Measured by: |new_variables| / |inherited_variables| > 0.

## M2M Constraint
When downstream is deterministic (Mode B forced), Aufhebung must produce exactly one actionable output.
Synthesis cannot output "undecidable" — it must escalate to KANSA via Rubicon Moment.
