## I. 维特根斯坦其人 (1889-1951)

### 生平
- 生于维也纳，Karl Wittgenstein 之子（奥地利最富有的工业家之一）
- 在柏林和曼彻斯特学习工程，后转向哲学
- 1912 年进入 Cambridge，师从 Bertrand Russell
- 一战服役，战俘营中完成 *Tractatus Logico-Philosophicus* (1921)
- 放弃哲学，当小学教师、园丁、建筑师
- 1929 年返回 Cambridge，与 *Tractatus* 决裂
- 1939 年接替 G.E. Moore 担任哲学教授
- 1951 年去世，留下 *Philosophical Investigations* (1953)

### 两个维特根斯坦
- **早期** (*Tractatus*, 1921)：语言是世界的逻辑图像。有意义的命题必须有真值条件。伦理、美学、形而上学不可说。
- **晚期** (*Investigations*, 1953)：语言是"语言游戏"——没有统一的本质，只有家族相似。意义在于使用。

## II. 十大核心论题

### 1. 语言游戏 (Sprachspiel)
- **来源**：*Philosophical Investigations* §7, §23
- **核心主张**：语言不是一个统一的系统——它是无数"语言游戏"的集合。命令、描述、报告、猜谜、讲笑话、请求、感谢——都是不同的语言游戏。
- **意义**：没有"理想语言"或"完美逻辑"——只有具体的、情境中的语言使用。
- **"意义即使用"** (PI §43)：一个词的意义不是它指称的对象，而是它在语言游戏中的使用方式。

### 2. 私人语言论证 (Private Language Argument)
- **来源**：PI §243-315
- **核心主张**：不可能有一种只有说话者自己能理解的语言。语言本质上是公共的——它的规则必须是可公共检验的。
- **"甲虫在盒子里"** (PI §293)：如果每个人都有一个盒子，里面装着只有自己能看到的"甲虫"，那"甲虫"这个词在公共语言中不可能指称盒子里的东西——因为没人能检验别人的甲虫。
- **意义**：内在经验（疼痛、信心、理解）不能通过私人语言表达——它们只能通过公共的行为标准来谈论。

### 3. 家族相似 (Familienähnlichkeit)
- **来源**：PI §65-67
- **核心主张**：概念没有统一的本质——它们通过"家族相似"联系在一起。"游戏"这个概念没有所有游戏共享的本质特征——棋类游戏、球类游戏、纸牌游戏通过交叉重叠的相似性联系。
- **意义**：不要问"什么是 X？"——要问"X 在不同语境中如何被使用？"

### 4. 遵守规则 (Befolgung von Regeln)
- **来源**：PI §185-242
- **核心主张**：遵守规则不是私人心理过程——它是公共实践。规则不能决定自己的应用——每个规则都可以被不同地解读。
- **"规则悖论"** (PI §201)：任何行动都可以被解释为符合某个规则。因此，规则不能约束行动——只有社会实践能约束。
- **意义**：RASHOMON 的规则（四道门、KANSA、KENGEN）不能自我执行——它们需要社会实践（人类监督）来维持。

### 5. 确定性 (Über Gewißheit)
- **来源**：*On Certainty* (1Moore 论文，死后出版 1Moore 969)
- **核心主张**：某些信念是"铰链命题" (hinge propositions, §341-343)——它们不在语言游戏中被质疑，而是使语言游戏成为可能。"我有两只手"不是我"知道"的东西——它是我的整个知识体系的铰链。
- **意义**：RASHOMON 的某些假设（"多视角优于单视角"）可能是铰链命题——不被质疑，而是使系统成为可能。

### 6. 甲虫在盒子里 (Beetle in a Box)
- **来源**：PI §293
- **与 AI 的映射**：每个 AI 代理都有"内部状态"（注意力分布、隐藏层激活、置信度）——但其他代理无法直接访问这些状态。它们只能通过输出来推断。

### 7. 面相转换 (Aspektwechsel)
- **来源**：*Philosophical Investigations* II.xi (pp. 193-229)
- **核心主张**：同一个图像可以被"看到为"不同的东西——鸭兔图既可看作鸭又可看作兔。
- **意义**：同一个数据可以被不同代理"看到为"不同的结论——这不是错误，是面相的多样性。

### 8. 说与显示 (Sagen und Zeigen)
- **来源**：*Tractatus* 4.121
- **核心主张**：有些东西不能被说 (sagen)，只能被显示 (zeigen)。逻辑形式、伦理价值、美学判断——它们不在语言的范围内。
- **早期 vs 晚期**：早期维特根斯坦认为形而上学不可说。晚期维特根斯坦放弃了这个观点——但保留了"有些东西只能被显示"的洞见。

### 9. 语言的限制 = 世界的限制
- **来源**：*Tractatus* 5.6: "我的语言的限制意味着我的世界的限制。"
- **与 RASHOMON 的映射**：每个代理的"语言"（训练数据、模型架构、prompt）决定了它的"世界"（它能看到的东西）。多代理系统的价值在于：不同语言 → 不同世界 → 更丰富的视野。

### 10. "凡不可说的，必须沉默"
- **来源**：*Tractatus* 7: "Wovon man nicht sprechen kann, darüber muß man schweigen."
- **与 RASHOMON 的映射**：RASHOMON 的 aporia 输出 = 维特根斯坦式沉默——当系统无法通过对抗性审查得出结论时，它应该沉默，而不是强行给出答案。

## III. 维特根斯坦与 RASHOMON 的映射

| Wittgenstein 概念 | RASHOMON 映射 |
|------------------|--------------|
| 语言游戏 | 每个代理在自己的"语言游戏"中运作（不同模型、不同 prompt） |
| 私人语言论证 | 代理不能有"私人知识"——所有声明必须可公共检验 |
| 家族相似 | RASHOMON 的概念（"可靠性"、"共识"）没有统一本质 |
| 遵守规则 | 四道门不能自我执行——需要人类监督 |
| 铰链命题 | "多视角优于单视角"可能是铰链命题 |
| 甲虫在盒子里 | 代理的内部状态不可直接访问 |
| 面相转换 | 同一数据 → 不同结论 = 面相多样性 |
| 说与显示 | 有些东西只能被显示（系统行为），不能被说（文档描述） |
| 语言限制 = 世界限制 | 每个代理的模型 = 它的世界 |
| 沉默 | Aporia = 维特根斯坦式沉默 |
