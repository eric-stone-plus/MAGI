# QUINTE R1 — 第一轮审计

> hm + CC + CW + OMP 四方独立审计

---

## HM（执政官）— 行政复核

**任务**: 从管理层视角审查报告的完整性和合规性。

**发现**:

1. **报告目的**: 巡视组要求说明运价波动原因，报告声称"运价波动均有合理市场依据"。结论正确与否取决于数据准确性。

2. **数据完整性**: 报告引用了 556 航次，台账有 555 有效行（接近）。但报告中 4 条航线的具体数据全部有误：
   - 运价数字截断（所有航线）
   - COA/现货分类错误（ADANG BAY、NEWCASTLE、KALIORANG）
   - KALIORANG 现货数据不存在

3. **管理层责任**: 报告由航运部编制，代表公司立场提交给巡视组。提交包含系统性错误的报告会损害公司信誉。管理层有责任确保提交材料的准确性。

4. **判定**: BLOCKED — 报告不得以当前版本提交。

---

## CC（合规审查）— 法律/合规风险

**任务**: 评估报告错误的法律和合规后果。

**发现**:

1. **《监察法》风险**: 巡视组依据《中国共产党巡视工作条例》开展工作。向巡视组提供虚假或失实材料，可能构成"对抗组织审查"。即使错误是无意的，"未尽到合理审查义务"本身就是过失。

2. **KALIORANG 问题定性**: 报告声称存在"现货.5-8.3"，台账中无此数据。这属于"提供不实信息"——无论是否有意，在巡视语境下性质严重。

3. **内部管控缺陷**: 从台账到报告的数据管道存在系统性故障，说明公司内部数据治理和报告审核机制有缺陷。巡视组可能据此要求公司加强内控。

4. **补救建议**: 
   - 立即撤回当前版本
   - 重新生成报告，附台账原始数据作为附件
   - 在报告中说明数据来源和核验方法
   - 建立报告提交前的数据核验流程

5. **判定**: BLOCKED — 存在合规风险，必须重写后附原始数据佐证。

## CW（代码/数据验证）— 技术审计

**任务**: 从数据角度验证报告的每一个数字。

**发现**:

1. **数字截断模式分析**:
   ```
   报告数字 → 台账实际 → 截断规律
   ".0"      → 12.11    → 12→丢, .11→.0(四舍五入?)
   ".1"      → 8.14     → 8→丢, .14→.1(截断)
   ".5"      → 6.88     → 6→丢, .88→.5(?)
   "0.4"     → 10.37    → 10→0?, .37→.4(四舍五入)
   "2.1"     → 10.78    → 10→丢?, .78→2.1(?)
   ```
   
   截断模式不完全一致——有些像四舍五入到小数点后一位，有些像直接丢弃整数部分。这可能意味着多个处理步骤叠加。

2. **COA 分类差异验证**:
   - ADANG BAY: 台账"长协"=36, "现货"=4。报告说 COA=25。差值 11。
   - 如果报告只算"中远海（长协）"=20 + "中远海"=8 = 28，仍不是 25。
   - 如果报告只算"中远海（长协）"=20 + "恒达"=5 = 25，那就对了！
   - **结论**: 报告可能只统计了特定船东（中远海长协 + 恒达）的航次，遗漏了海昌、德运、时代等船东的 COA 航次。

3. **NEWCASTLE 分类验证**:
   - 台账: 长协=30, 小长协=4, 现货=28。报告说 COA=~45, 现货=~16。
   - 30+4=34 ≠ 45。28 ≠ 16。
   - 如果报告把"长协+小长协+部分现货"都算作 COA，可能得到 ~45。但这分类逻辑有问题。

4. **判定**: BLOCKED — 数据错误已被技术验证确认。报告需要从台账原始数据重新生成。

## OMP（台账勾稽）— 业务/财务审计

**任务**: 从台账数据本身验证报告的业务逻辑。

**发现**:

1. **运价合理性**: 虽然报告的数字有误，但台账中的运价本身是否合理？
   - ADANG BAY COA: 3.96~12.11 USD/吨，对应 Supramax/Handysize 船型，印尼→华南航线，2022-2026年。市场参考：同期 BDI 从 500 到 3400，Supramax TCE 约 $5,000-30,000/天。换算为运价约 $5-15/吨。台账数据在合理范围内。
   - NEWCASTLE COA: 10.37~20.38 USD/吨，澳洲→华南，Panamax 船型。市场参考：同期 Panamax TCE 约 $8,000-25,000/天，换算约 $10-20/吨。台账数据在合理范围内。
   - **结论**: 台账中的运价本身是合理的，没有明显异常偏高。

2. **COA vs 现货价差**: 
   - ADANG BAY: COA 均价 ~6.9, 现货均价 ~8.3, COA 低 17%。合理。
   - NEWCASTLE: COA 均价 ~15.2, 现货均价 ~13.8, COA 高 10%。**异常**——COA 应低于现货。原因：4 个小长协（15.07~16.33）拉高了 COA 均价。如果排除小长协，仅看"长协"（30 航次），均价可能低于现货。
   - TABONEO: COA 均价 ~8.1, 现货均价 ~8.1, 几乎无差异。说明该航线 COA 没有价格优势。

3. **报告核心结论的业务验证**: "未发现运价异常偏高"——从台账数据看，运价确实与市场一致，没有明显偏高。但这个结论需要用正确的数据重新论证。

4. **判定**: BLOCKED — 结论方向可能正确，但论证过程基于错误数据，不可信。需要用正确数据重新论证。
