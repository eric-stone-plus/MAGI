Architecture

Dialectical Spiral Protocol
THE PROTOCOL

Five phases of
convergent inquiry

Each phase serves a distinct purpose in the journey from raw question to grounded verdict. The phases follow the Magi's journey — from the star that guides them, through the road that tests them, to the manger where they converge.

PHASE 0

The Star 星

星の導き

The Magi followed a star. hm follows the evidence — decomposing the problem, gathering data, and assigning each Gift to the agent best suited to carry it. The only interactive phase: if the problem is ambiguous, hm clarifies before proceeding.

PHASE 1

The Offering 獻禮

三博士の獻禮

Three parallel analyses, each through a distinct epistemological lens. Gold verifies facts, Frankincense synthesizes meaning, Myrrh maps risks. All three run concurrently — independent perspectives on the same evidence.

PHASE 2

The Journey 旅路

辯證の旅路

The dialectical spiral. Each Gift reviews the previous Gift's output through its own lens — Gold verifies Myrrh's risks, Frankincense synthesizes Gold's facts, Myrrh stress-tests Frankincense's patterns. Directed, not random. O(n), not O(n²).

PHASE 3

The Manger 馬槽

収束の馬槽

Have the Magi converged? hm evaluates three dimensions — factual agreement (0.4), interpretive consensus (0.3), and risk resolution (0.3). Score ≥0.7 → proceed. Below → targeted restart on the weakest dimension.

PHASE 4

The Revelation 啟示

最終の啟示

The final synthesis. Not a binary verdict, but a confidence topology — a map of what we know, how well we know it, and where the knowledge breaks down. Known unknowns explicitly documented. Dissent preserved, not suppressed.

DIALECTICAL SPIRAL

Not cross-review.
Spiral review.

QUINTE's cross-review: every agent reviews everyone — O(n²), flat, undifferentiated. MAGI's spiral: each Gift reviews the previous — O(n), directed, narrative arc.
